# Welcome to the wiki!

GodMode is a 75/75 Custom Content FFXI Private server.

Exceed the limits and obtain GodHood.

# Server Information

- --server godmode.ddns.net
- 75/75 cap
- Treasures of Aht Urgan content level
- FoV enabled
- GoV enabled
- Survival Guide Teleports Enabled
- Normal Limit Breaks
- No Subjobs or advanced jobs until your first Rebirth.
- Normal exp rates
- Trusts Enabled but limited to 5 (as you Rebirth the amount simultaneously allowed will reduce)
    - Ferreous Coffin
    - Valaineral
    - Shantotto II
    - Rainmard
    - Monberaux
- Move speed is doubled.
- Craft rates are 9x
- All crafts can go max.
- Fishing is on, but probably doesn't work completely.
- Era Treasure Hunter style.
- Lotteries are set to always spawn.
- Ah bot enabled

# Player Hub

Players can use the !hub command to be teleported to a utility hub.
- Homepoint / Survival Guide 
- Exp camp fast travel book
- Cheat Code book that will do the following:
    - Cap skills
    - Add all spells
    - Add all Songs
    - Add Cor Rolls
    - Add some PUP attachments
    - Add all mounts
    - Add all Chairs
    - Unlock all Weaponskills
    - Complete Rise of the Zilart
    - Complete COP
    - Set Rank to 10

# Main Story Quest

On GodMode there is a narrative driven Main Story Quest (MSQ)

Phoenix in Valhalla (Heaven) has chosen you to be a vessel of power.
